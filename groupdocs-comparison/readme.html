Installation

  1. Create a snippet called GroupDocsComparison, and paste in the contents of snippet.src.php
  2. Insert [[GroupDocsComparison? &form=`xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`]] where you would like the document to appear

    3. As a minimum, the snippet requires a GroupDocs Form ID
